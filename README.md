# One-Mirror

> This is a Student Portal for Restart Network where the student can view their profile and 
the admin can register student and edit students data once the student sends a request to the admin.

> I personally worked on Admin Panel frontend and backend together with my other colleague.  
 

### Technologies Used

* MongoDB
* React
* ExpressJs
* NodeJS
* Bootstrap

### Challenges 

* Securing the website
